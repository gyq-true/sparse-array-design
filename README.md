# 基于迭代凸优化的平面稀疏阵列综合算法 (Robust IDEA)

## 📌 项目简介

本项目实现了一种鲁棒的、用于天线阵列综合的迭代优化算法。本代码基于扩展的 IDEA（Iterative Deflation and Expansion Algorithm）框架构建，引入了加权范数松弛与自适应空间搜索策略，旨在解决阵列天线设计中极具挑战性的多维非凸优化问题。

**适用范围：** 本项目严格基于**迭代凸优化技术路线**，专门针对**非均匀间隔、非均匀激励的大规模平面稀疏阵列设计**。适用于需要同时降低天线系统硬件成本（减少阵元数量、降低互耦）并维持高辐射性能（如窄波束宽度、低峰值旁瓣电平、高主瓣增益）的工程场景。

---

## ⚙️ 核心算法原理

平面稀疏阵列的综合本质上是一个 NP-hard 的高度非凸问题（位置与激励的联合优化）。本算法摒弃了容易陷入局部最优的传统全局启发式算法，采用**迭代凸优化（Iterative Convex Optimization）**路线。

核心思想是通过交替执行“在连续空间内扩展候选位置”与“在约束条件下压缩冗余阵元”的操作，将原非凸问题转化为一系列可求解的凸逼近子问题。此外，算法引入了**加权 L1.5 范数（Reweighted L1.5-norm）**作为稀疏促进项，相比传统的 L1 范数，L1.5 范数在阵元剔除与能量保持之间取得了更好的平衡。

---

## 🔄 算法执行步骤

算法的完整运行周期包含以下关键步骤：

### 1. 物理约束与启发式初始化 (Initialization)
* **随机扰动网格**：打破传统均匀半波长网格的限制，向初始均匀网格中注入受控的随机坐标抖动（Jitter），提供更丰富的拓扑空间。
* **密度锥化预处理**：利用余弦锥化（Cosine Tapering）概率分布模型进行初始阵列的降维，直接舍弃边缘低贡献区域的冗余阵元，极大降低凸优化求解器的计算维度。
* **边界锁定**：提取阵列最外围的顶点进行物理位置锁定，确保在稀疏化过程中阵列的总有效孔径（Aperture）不发生收缩，从而保证主瓣宽度（HPBW）。

### 2. 加权稀疏化优化 (Sparsification)
* 在给定的目标峰值旁瓣电平（PSLL）约束下，以最小化**加权 L1.5 范数**与 **L2 能量平滑项**的混合代价函数为目标，调用 CVX 凸优化求解器对当前阵元激励进行优化。
* 利用上一次迭代的激励幅度构建对角权重矩阵（幅度越小，惩罚权重越大），迫使对方向图贡献微弱的阵元激励加速趋近于零。

### 3. 自适应空间膨胀 (Inflation)
* 为了寻找更优的非均匀物理位置，围绕当前存活的每一个阵元（锁定点除外），在其局部邻域内按自适应计算的移动半径（随迭代进程动态缩小）生成 $P$ 个虚拟候选顶点。

### 4. 群组稀疏优化 (Group Optimization)
* 将同一个父阵元分裂出的 $P$ 个虚拟顶点视为一个“群组”。
* 构建基于**群组 L1.5 范数（Group L1.5-norm）**的新一轮凸优化模型。优化过程会在满足旁瓣约束的条件下，在每个群组内部倾向于只保留一个最优位置的激励。

### 5. 阵元收缩与状态更新 (Deflation & Update)
* **质心法聚合**：根据群组内各个虚拟顶点的优化后激励幅度，利用质心加权公式（Center of Mass）计算出新的、偏离原网格的连续物理坐标 $(x, y)$。
* **硬阈值剔除**：将计算出的新激励与预设的剔除阈值（Threshold）进行比较，彻底删除低于阈值的阵元。

### 6. 动态扰动与收敛判定 (Dynamic Perturbation)
* 为防止算法陷入“死锁”的局部极小值，每隔固定迭代轮次，主动向当前阵列的坐标和激励幅相中注入微小的随机抖动噪声。
* 当阵元总数连续多个迭代周期保持不变时，判定算法收敛，跳出主循环并输出最终的**非均匀间隔坐标**与**非均匀激励幅度**。

## 🛠️ 依赖项与环境要求

* **MATLAB**: 推荐使用 R2019b 及以上版本，以支持相关的向量化寻峰及矩阵运算。
* **CVX 工具箱**: 必须安装并配置好 CVX (面向 MATLAB 的凸优化建模系统)。求解器建议使用 `Mosek` 或 `SeDuMi` 以处理大规模复数变量。运行前请确保 `cvx_setup` 已正确执行。
